# EXIF Metadata Extractor (Interactive CLI)

A Python script to extract EXIF metadata from images with an interactive menu.

## Features
- **Interactive menu** to select an image.
- Extracts **camera model, timestamp, GPS location**.
- Generates **Google Maps link** for location.
- Works on **Kali Linux terminal**.
